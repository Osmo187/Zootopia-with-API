# Zootopia with API

A Python project that generates an animal encyclopedia website by fetching data from the API Ninjas Animals API.

## Installation

Clone the repository and install the dependencies:

    pip install -r requirements.txt

Create a `.env` file in the root directory with your API key:

    API_KEY='your_api_key_here'

## Usage

Run the following command and enter an animal name when prompted:

    python3 animals_web_generator.py

The website will be generated as `animals.html`.
